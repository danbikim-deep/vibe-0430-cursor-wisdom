# 명언 생성기

브라우저에서 동작하는 가벼운 명언 앱입니다. 기본 명언을 랜덤으로 보여주고, 사용자가 직접 명언을 추가/삭제하며, 복사 및 공유 기능을 사용할 수 있습니다.

## 지금까지 구현된 내용

- 기본 명언 20개 내장 및 랜덤 노출
- 같은 명언이 연속으로 반복될 가능성을 줄인 랜덤 선택 로직
- 사용자 명언 추가 기능(문구/저자 입력)
- 사용자 명언 삭제 기능(기본 명언은 삭제 불가)
- 클립보드 복사 기능
- Web Share API 기반 공유 및 트위터 공유 fallback
- 라이트/다크 테마 전환 및 테마 상태 저장
- 상태 메시지 안내(저장/복사/공유/오류 등)
- 명언 전환 시 짧은 페이드 애니메이션
- 반응형 UI(모바일/데스크톱 2단 레이아웃)
- 키보드 접근성 강화(Enter/Space로 버튼 활성화)
- `aria-live`, `role="status"` 등을 활용한 접근성 개선
- `prefers-reduced-motion` 대응

## 데이터 저장 방식

- 사용자 명언은 `localStorage`의 `quotes_user_v1` 키에 저장됩니다.
- 테마 설정은 `localStorage`의 `quote_theme_v1` 키에 저장됩니다.
- 저장 데이터가 없거나 손상된 경우를 대비한 방어 로직이 포함되어 있습니다.

## 입력 검증 규칙

- 문구와 저자를 모두 입력해야 저장 가능
- 문구 최대 180자, 저자 최대 40자
- 앞뒤 공백 제거 및 연속 공백 정규화
- 문구+저자 조합 중복 등록 방지

## 파일 구성

- `index.html`: 앱 마크업 구조, 버튼/폼/접근성 속성 정의
- `style.css`: 색상 테마, 레이아웃, 애니메이션, 반응형 스타일
- `app.js`: 명언 로직, 저장소 처리, 이벤트 핸들러, 테마/공유/복사 기능

## 실행 방법

별도 빌드 없이 정적 파일로 동작합니다.

1. 프로젝트 루트에서 `index.html`을 브라우저로 열기
2. 또는 Live Server 같은 정적 서버로 실행
